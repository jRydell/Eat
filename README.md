# Eat
## -Whats for dinner?

An app that both helps you create your weekly menu, shopping list and how to cook the meals.



## Get started

1. Clone repository
   ```bash
   git clone https://github.com/jRydell/Eat
   ```
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
    npx expo start
   ```
